# WiKi4DCube Name Policy (Brand / Identity)

This repository is released under the MIT License.  
However, **the name “WiKi4DCube” and related brand identifiers are controlled** to protect users from confusion and to preserve a single canonical specification source.

## 1. Purpose

This policy exists to ensure:

- Users can identify the **canonical** WiKi4DCube specification and releases.
- Forks and modified versions do not misrepresent themselves as official.
- The ecosystem can grow while maintaining a stable reference point.

## 2. What is covered

The following identifiers are covered by this policy:

- The name: **WiKi4DCube**
- “WiKi4DCube Spec”, “WiKi4DCube Canonical”, “WiKi4DCube Official”
- Any future official logos or badges published by the maintainer (**gk0729**)
- Phrases implying official status: “official”, “certified”, “canonical”, “reference implementation” (when used in a confusing way)

## 3. What you may do (allowed)

You may:

- Fork this repository and modify the code/spec.
- Build implementations compatible with the spec.
- Publish research and tools that support the WiKi4DCube dimension space.
- State factual compatibility claims such as:
  - “Compatible with WiKi4DCube Spec v0.1”
  - “Implements WiKi4DCube 241-dim coordinate definitions”

## 4. What you may NOT do (restricted)

Without explicit written permission from **gk0729**, you may NOT:

- Name your fork or product “WiKi4DCube” in a way that implies it is official.
- Use “WiKi4DCube Official”, “WiKi4DCube Canonical”, or similar wording.
- Use an official WiKi4DCube logo/badge (if any) to imply endorsement.
- Claim to be the official WiKi4DCube release channel.

## 5. How to request permission

If you need permission to use the WiKi4DCube name for a fork, distribution, or product:

- Open a GitHub issue in this repository describing your use case; or
- Contact the maintainer (gk0729) via the contact method listed in README.

## 6. Disclaimer

This Name Policy is intended as a **community clarity / anti-confusion policy**.
It does not replace formal trademark registration.
If formal trademarks are registered in the future, this policy may be updated to reflect that.
